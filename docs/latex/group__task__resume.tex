\begin{DoxyPre}void task\_resume( task\_t xTaskToResume );\end{DoxyPre}


INCLUDE\+\_\+v\+Task\+Suspend must be defined as 1 for this function to be available. See the configuration section for more information.

Resumes a suspended task.

A task that has been suspended by one or more calls to task\+\_\+suspend () will be made available for running again by a single call to task\+\_\+resume ().

{\em x\+Task\+To\+Resume} & Handle to the task being readied.\\
\hline
\end{DoxyParams}
